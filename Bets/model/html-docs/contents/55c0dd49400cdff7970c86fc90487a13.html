CancelBet

UMLUseCase
Bet&Ruin :: Use Case Diagram :: CancelBet

Description

Event flow

Main flow

  1. User clicks on the Cancel bet in the BetInfoGUI
  2. System removes the bet from the Registered bets of the User
  3. System disables the Cancel bet in the BetInfoGUI
  4. System adds 75% of the money form the bet into the Walet of the User

Precondition

  • User must be logged in
  • User must have at least one bet registered
